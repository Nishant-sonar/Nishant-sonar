### 📈 Global Superstore Sales Forecasting

<div align="center">

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![Pandas](https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white)

![Kaggle](https://img.shields.io/badge/Kaggle-035a7d?style=flat-square&logo=kaggle&logoColor=white)

</div>

> 📊 End-to-end **time series forecasting** on 4 years of retail data — from raw EDA to SARIMA deployment.

| metric | value |
|--------|-------|
| 📅 Data Span | **2015 – 2018** |
| 🗂️ Records Analyzed | **9,800** |
| 📐 Features Explored | **18** |
| 🎯 Forecast Accuracy | **19% MAPE** |

**What it does:**
- ✅ ADF stationarity test · ACF/PACF analysis · Seasonal decomposition
- ✅ IQR-based outlier handling — replaced with monthly averages
- ✅ Auto SARIMA via pmdarima — best model selected automatically
- ✅ Uncovered: seasonal peaks, West+East = 35% of total revenue
- ✅ Interactive Plotly dashboards for inventory & marketing decisions
